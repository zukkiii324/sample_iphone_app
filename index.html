カテゴリごとに拡張できる静的サイトの土台

シンプルなホームページに加えて、用途別のサブページをカテゴリごとに整理したフロントエンド構成です。 それぞれのページが独立して開発しやすいようにファイルを分割し、スケールするウェブ体験を実現します。

サブページを見る 構成を確認する

このサイトについて

このテンプレートは、ホームページと複数のカテゴリに紐付いたサブページをセットで構築するための土台です。 デザインシステムや UI パターンを共通化しつつ、各カテゴリページごとに独自のスタイルやコンテンツを展開できます。

共通デザイン

assets/css/base.css に共通スタイルを集約し、ブランドイメージを保ったまま全ページに適用します。

カテゴリ別拡張

サブページはカテゴリごとにディレクトリを分割。各ディレクトリ内で個別の CSS やアセットを追加できます。

保守しやすい構成

ナビゲーションやパンくずリストなどのパターンを再利用しながら、用途に合わせたページ開発が可能です。

カテゴリ別サブページ

それぞれのカテゴリは独立したフォルダに分かれており、個別の開発・公開フローを想定しています。必要に応じて新しいカテゴリを追加し、柔軟にスケールできます。

フロントエンドファイル構成

ホームとサブページが相互に干渉しないよう、カテゴリごとにフォルダを分割しています。共通スタイルだけを共有し、その他のアセットは各カテゴリ内に配置できます。 

sample_iphone_app/
├── index.html                 # ホームページ
├── assets/
│   └── css/
│       └── base.css           # 全ページ共通スタイル
├── pages/
│   ├── business/
│   │   ├── index.html         # ビジネスカテゴリのトップ
│   │   └── styles.css         # ビジネス専用スタイル
│   ├── creative/
│   │   ├── index.html         # クリエイティブカテゴリのトップ
│   │   └── styles.css         # クリエイティブ専用スタイル
│   ├── support/
│   │   ├── index.html         # サポートカテゴリのトップ
│   │   └── styles.css         # サポート専用スタイル
│   └── lifestyle/
│       ├── index.html         # ファミリー＆ライフカテゴリのトップ
│       └── styles.css         # ファミリー＆ライフ専用スタイル
└── docs/…                     # デプロイ手順などの補足資料
